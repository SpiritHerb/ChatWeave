# ChatWeave

ChatWeave is a local-first, browser-based conversation migration tool focused on exporting and merging chat history data into RikkaHub-compatible backup packages.

## Features

- Runs locally in the browser.
- Supports importing from multiple data sources.
- Helps merge selected conversations into a target backup package.
- Provides pre-check and download flow before final output.

## Usage

1. Open `index.html` in a modern browser.
2. Import source data.
3. Select conversations and merge options.
4. Provide target backup ZIP and generate output.

## Privacy

This repository intentionally contains code only.
No personal chat exports, backup ZIPs, databases, or logs are included.
